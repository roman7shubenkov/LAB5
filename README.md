# LAB5  Графы

### 1.13 Рефлексивный, антирефлексивный, частично рефлексивный орграфы

Орграф, или ориентированный граф, — это граф, рёбрам которого присвоено направление.

Рефлексивный орграф — это орграф, в котором каждая вершина имеет петлю (ребро, соединяющее вершину с самой собой). Антирефлексивный не имеет таких петель. Частично рефлексивный - орграф у которого как минимум одна вершина с петлёй и как минимум одна без.
